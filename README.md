# 🕷️ Spider-Net Dashboard

Um dashboard temático inspirado no universo do Homem-Aranha, desenvolvido para praticar conceitos de desenvolvimento front-end moderno utilizando Tailwind CSS e DaisyUI. A aplicação simula um painel de monitoramento de atividades heroicas, exibindo informações do herói, status do traje e alertas de crimes recentes.

## 🚀 Tecnologias Utilizadas

* **HTML5**
* **Node.js**
* **Tailwind CSS**
* **DaisyUI**
* **CSS Utility-First Design**

## 📋 Funcionalidades

* Interface responsiva para diferentes tamanhos de tela.
* Navbar estilizada com status do sistema em tempo real.
* Card de informações do herói com indicadores de recursos.
* Sistema de alertas utilizando componentes expansíveis (Collapse/Accordion).
* Layout moderno utilizando CSS Grid.
* Tema escuro inspirado em sistemas de monitoramento e segurança.

## ⚙️ Instalação

Clone o repositório:

```bash
git clone https://github.com/gapashi/spider-painel.git
```

Acesse a pasta do projeto:

```bash
cd spider-net-dashboard
```

Instale as dependências:

```bash
npm install
```

## ▶️ Executando o Projeto

Inicie o ambiente de desenvolvimento:

```bash
npm run dev
```

O projeto estará disponível em:

```text
http://localhost:3000
```

*(A porta pode variar conforme a configuração do ambiente.)*

## 🎨 Bibliotecas de Interface

### Tailwind CSS

Framework utilitário utilizado para construção rápida da interface através de classes de estilo diretamente no HTML.

### DaisyUI

Biblioteca de componentes baseada em Tailwind CSS, utilizada para criar elementos visuais como:

* Navbar
* Cards
* Badges
* Buttons
* Stats
* Collapse (Accordion)

## 📚 Objetivo do Projeto

Este projeto foi desenvolvido com fins educacionais para praticar:

* Responsividade com Tailwind CSS
* Utilização de componentes do DaisyUI
* Estruturação de layouts com Grid e Flexbox
* Organização visual de dashboards
* Desenvolvimento front-end utilizando Node.js como ambiente de execução
